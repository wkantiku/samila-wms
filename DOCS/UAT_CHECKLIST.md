## Module ที่ต้องทดสอบ

### 1. Login / Authentication
- [ ] เข้าสู่ระบบด้วย admin ได้
- [ ] เข้าสู่ระบบด้วย manager ได้
- [ ] กรอก password ผิด → แสดง error
- [ ] ออกจากระบบได้

### 2. เลือกคลังสินค้า (Page 2)
- [ ] เห็นรายชื่อคลังสินค้าครบ
- [ ] แสดงชื่อ Company ที่กล่องคลัง
- [ ] คลังที่ปิดใช้งาน → กดไม่ได้
- [ ] กดเลือกคลัง → เข้า Dashboard ได้

### 3. Dashboard
- [ ] แสดง KPI summary ถูกต้อง
- [ ] กราฟแสดงผลได้
- [ ] เปลี่ยนภาษา TH/EN ได้

### 4. รับสินค้า (Receiving)
- [ ] เพิ่มรายการรับสินค้าได้
- [ ] สแกน Barcode / QR Code ได้
- [ ] บันทึกข้อมูลได้

### 5. คลังสินค้า (Inventory)
- [ ] ค้นหาสินค้าได้
- [ ] แสดงรายการสินค้าถูกต้อง
- [ ] Export Excel/PDF ได้

### 6. สินค้า (Product)
- [ ] เพิ่มสินค้าใหม่ได้
- [ ] แก้ไขข้อมูลสินค้าได้
- [ ] ลบสินค้าได้

### 7. หยิบสินค้า (Picking)
- [ ] สร้าง Picking Order ได้
- [ ] อัปเดตสถานะได้

### 8. จัดส่ง (Shipping)
- [ ] สร้าง Shipment ได้
- [ ] พิมพ์ใบส่งของได้

### 9. รายงาน (Reports)
- [ ] ดูรายงานสรุปได้
- [ ] Export ได้

### 10. ผู้ใช้งาน (Users) — Admin only
- [ ] เพิ่มผู้ใช้ได้
- [ ] กำหนด Role ได้
- [ ] Assign คลังสินค้าได้

### 11. Mobile / PWA
- [ ] เปิดบน Chrome Android ได้
- [ ] "Add to Home screen" ได้
- [ ] หน้าจอ Responsive ดีบน mobile

## ปัญหาที่พบ (Bug Report)

| # | Module | คำอธิบาย | ความรุนแรง | สถานะ |
|---|---|---|---|---|
| 1 | | | Critical / High / Medium / Low | Open |
| 2 | | | | |
| 3 | | | | |

**ระดับความรุนแรง:**
- 🔴 **Critical** — ใช้งานไม่ได้เลย
- 🟠 **High** — ฟีเจอร์หลักทำงานผิด
- 🟡 **Medium** — ผลกระทบต่อการใช้งาน
- 🟢 **Low** — UI/UX ปรับปรุงได้
