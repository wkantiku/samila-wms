## 📱 Mobile Shortcuts

### **iPhone/iPad (iOS)**

**ตัวเลือก 1: PWA (ง่ายสุด)**
```
1. เปิด Safari → http://your-server:3000
2. Tap "Share" → "Add to Home Screen"
3. ตั้งชื่อ: SAMILA WMS
4. ตั้ง Icon: ใช้ samila_logo.png
5. Tap "Add"
```

**ตัวเลือก 2: Shortcuts App**
```
1. เปิด Shortcuts App
2. Tap "+" → Create Shortcut
3. Add Action: Open URLs
4. URL: http://your-server:3000
5. ตั้งชื่อ: SAMILA WMS
6. เลือก Icon: samila_logo.png
7. Save & Add to Home Screen
```

```
3. ตั้งชื่อ: SAMILA WMS
4. ตั้ง Icon: samila_logo.png
5. Tap "Install"
```
